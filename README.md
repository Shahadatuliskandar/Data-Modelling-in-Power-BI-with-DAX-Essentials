# Data Modelling in Power BI with DAX Essentials
Please follow these steps to start using the sample data properly:
  1. Click on the CSV Data Sources.zip filename from the list
  2. Click on the Raw button and your web browser should be automatically downloading the file into your system
  3. Open the downloaded zip file
  4. Extract all contents into your own folder location. Make sure you have extracted these items:
       (a) All Employees Record.csv
       (b) Ex-Employees Record.csv
       (c) Geographic Details.csv
       (d) Products Details.csv
       (e) Training Programs.csv
       (f) Training Records.csv
  6. Make sure all the items above are extracted in a new single folder location
  7. OPTIONAL: Delete the zip file to avoid any confusion
  8. Click on the Excel Data Sources.zip filename from the list
  9. Click on the Raw button and your web browser should be automatically downloading the file into your system
  10. Open the downloaded zip file
  11. Extract all contents into your own folder location. Make sure you have extracted these items:
       (a) Group Work folder
       (d) Data_File - East Malaysia.xlsx
       (e) Data_File - North.xlsx
       (f) Data_File - Wilayah Persekutuan.xlsx
  12. Make sure all the items above are extracted in a new single folder location
  13. OPTIONAL: Delete the zip file to avoid any confusion
